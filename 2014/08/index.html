How to change Mac OS Dashboard Background.

How to change Mac OS Dashboard Background.

  1. Find the background you want to use as Dashboard wallpaper, duplicate and rename it to PNG using Preview’s Export tool,
    name it ‘pirelli.png’
  2. Right click on Finder, then click “Go To Folder” function and type the following directory path :-
    Code:
    /System/Library/CoreServices/Dock.app/Contents/Resources/
  3. Search for ‘pirelli.png’ and copy to some safe place (incase you want your old background) or delete it.
  4. Now copy your own image of ‘pirelli.png’ into the folder.
  5. Open Terminal and run this command :-
    Code:
    killall Dock
  6. Now you have a new background on Dashboard

How to create custom DMG file archives

Step 1: Open Disk Utility.app from your /Applications/Utilities/ folder.
Press “New Image”.

Step 2: Pick a name and size for your archive and then save it somewhere you’ll remember.
(Preferably your Desktop.)

Step 3: Open up your newly created DMG file.

Step 4: Drag the sidebar all the way to the left until you can’t see it anymore.

Step 5: Right click/CTRL just below the name of the archive and select “Hide Toolbar”.

Step 6: Now Right click/CTRL click inside of the folder and choose “Hide View Options”.

Step 7: Change the settings as in the picture below. (Example background image can be downloaded from HERE.)

Step 8: Now drag the file/application you want the user to “install” onto the archive, together with an alias of the Application folder.

Step 9: Finally, open up Disk Utility once again and hit the “convert” button.

Step 10: DONE!
You should now have a final DMG archive, ready to be distributed.
